# Changelog

All notable changes to the Pinable SDK will be documented in this file.

## [8.1.1] - 2026-05-06

### Added
- **Ministop Specific Support**: Added dedicated Android and iOS sample apps for Ministop app vendors.
- **SDK Data Identification**: Implemented sample code and guides to help host apps identify SDK data in two modes:
    - **Simple Mode**: For basic data exposure.
    - **Normal Mode**: For full SDK data interaction and lifecycle management.
- Updated integration documentation to clarify the distinction between these modes for the Ministop environment.

## [8.1.1] - 2026-03-16

### Added
- Improved beacon scanning performance.
- New notification handling logic for Android 14.
- Enhanced logging for debugging SDK initialization.

### Fixed
- Fixed a crash when the SDK is initialized without a valid API key.
- Resolved UI glitches in the offer popup on small screen devices.
